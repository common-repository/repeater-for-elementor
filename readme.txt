[youtube https://www.youtube.com/watch?v=C-xoIp7lIAY]

**DEMO**: <https://demo.add-ons.org/demo-repeater-fields/>

The Repeater Fields for Elementor Forms allow you to create one or more sets of fields that can be repeated.

Have you ever wanted to let your users submit multiple entries of the same field set as a single form on your WordPress site? If so, you’re in luck! This is a plugin to help you do it!

== Features ==
- **File Uploads Support**
- **Minimum Rows**: Sets a limit on how many rows of data are required.
- **Maximum Rows**: Sets a limit on how many rows of data are allowed.
- **Button Label**: The text shown in the ‘Add Row’ button.
- **Conditional Logic support**: The field conditional Logic support
- **Date picker support**: The field date/time support
- **Entry and print preview support**
- **Payment fields support**
- **Drag and drop repeatable fields**

== Installation ==
**Normal installation**

1. Download the repeater-field-for-elementor.zip file to your computer.
2. Unzip the file.
3. Upload the `repeater-field-for-elementor` directory to your `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Document: https://add-ons.org/document-elementor-foms-repeater-field/
